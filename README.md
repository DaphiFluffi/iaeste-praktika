## How to Execute

1. Download the repository and install python, open VSCode
2. Import a .csv file in the IAESTE AC Format from Google Sheets specifically
3. Go to ´convert_csv_to_json.py´ and adjust the .csv name in line 5
4. Execute ´python convert_csv_to_json.py´. You will get a short Success Message in the Terminal (praktika.json erfolgreich erstellt! Anzahl Einträge: 41)
5. Execute ´python -m http.server´ and go to localhost:8000 to view
6. Fork this repository and push your changes
7. Host on Github Pages or Github Pages with a custom domain. Source: ´\main´
